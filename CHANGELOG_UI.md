# CHANGELOG UI

## Что изменено

### Глобально
- Введён единый дизайн-язык через токены: `miniapp/tokens.css`.
- Полностью обновлён UI-стиль в dark layered эстетике (Apple clean + Spotify): `miniapp/styles.css`.
- Добавлен Telegram theme bridge с реакцией на смену темы: `miniapp/app.js`.
- Добавлены универсальные UI helper-компоненты для списков/состояний: `createListRow`, `renderEmpty`, `renderError`, `renderSkeletonList`.

### По экранам
- `mode`
  - Обновлены тексты и визуальная иерархия выбора режима.
  - Карточки стали крупнее, с более чистым CTA.

- `dashboard`
  - Портфель/позиции переведены на единый list-row стиль.
  - Добавлены skeleton-состояния загрузки.

- `trade`
  - Обновлены CTA и визуальная плотность формы.
  - Поля и поиск приведены к единой компонентной системе.

- `lookup`
  - Упрощён CTA, улучшена читаемость карточки результата.

- `movers`
  - Списки лидеров роста/падения в едином компонентном стиле.

- `alerts`
  - Список алертов и кнопки действий переведены в новый row-pattern.
  - Empty/error/loading состояния унифицированы.

- `more`
  - Блоки USD/RUB, импорт, статьи, очистка портфеля приведены к новой системе карточек и кнопок.

- `budget` (overview/income/expenses/funds/savings/close/settings)
  - Все карточки, формы, чипы, переключатели и списки обновлены под новую визуальную систему.
  - Сохранена функциональность онбординга, целей, расходов, накоплений и закрытия месяца.

## Микрокопирайт
- Сокращены и упрощены основные заголовки и CTA в `index.html`.
- Унифицированы product-state формулировки в JS (`Пока пусто`, `Не удалось загрузить`, и т.д.).

## Технически
- Бизнес-логика, API, контракты и роутинг не менялись.
- Изменён только UI-слой (`index.html`, `styles`, UI helper rendering в `app.js`).

## Изменённые файлы
- `miniapp/index.html`
- `miniapp/tokens.css` (новый)
- `miniapp/styles.css`
- `miniapp/app.js`
- `DESIGN_SYSTEM.md` (новый)
- `CHANGELOG_UI.md` (новый)

## Скриншоты
- В этой среде скриншоты не генерировались автоматически.
- Для проверки экранов: откройте Mini App в Telegram и последовательно пройдите вкладки `mode`, `dashboard`, `trade`, `lookup`, `movers`, `alerts`, `more`, а также бюджетные вкладки.
